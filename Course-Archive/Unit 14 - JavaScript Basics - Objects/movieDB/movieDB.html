Basic Movie Database.

Create an array of movie objects. Each will have a title, rating and hasViewed property. Iterate through the array to print out in form:

You have watched "Les Miserables" - 5 stars

  • "add" - adds new movie to the database
  • "list" - lists all items in the database
  • "delete" - deletes a selected movie
  • "quit" - quits the app
